ACT Art Finder

A free web and iOS application created by Command-X (Canberra Grammar School Code Cadets) that gives the location of public art and arts facilities in the ACT.



About ACT Art Finder

ACT Art Finder is a web and iOS application created by Command-X during GovHack 2014 . It gives the location of arts facilities and exhibits in the ACT.

It is designed to to help Canberra's citizens and tourists appreciate all of the art facilities and exhibits around the nation's capital. It includes data on over 120 beautiful art exhibits and 13 art facilities, giving the user a great view of where art is near them. The data is displayed on a Google Map so as to provide a useful, well-designed interface to find information about art in the ACT.

Download the source code for iOS from GitHub here , which includes an "I've been there" list that tracks which artworks you have visited. In the future, we plan to add in a location feature that displays your current location, as well as a display that shows how far away art is from you and what is closest to you.

Scroll down to "Data" for the basic web version of this app.
